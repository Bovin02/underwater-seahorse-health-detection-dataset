# Underwater Seahorse Health Detection Dataset

This repository provides the dataset used for underwater seahorse health detection.

## Dataset Description

The dataset contains underwater images of seahorses collected from real aquaculture environments. Each seahorse instance is annotated according to its health status based on tail posture.

The health categories include:

- **Normal**
- **Bacterial enteritis**
